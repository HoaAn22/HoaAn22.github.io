Bin mi trng (Environment Variables)

Bin mi trng l mt kin thc quan trng cho thc hin mt da n c cha cc thng tin quan trng nhy cm nh API Key, ID, Mt khu,... Vic thit lp bin mi trng gi tr ch trong my m khng lan truyn ra ngoi, tng cng kh nng bo mt.

C 2 cc chnh:

  • To bin mi trng lu trong windows
  • dng file .env (Local Environment File)

To bin mi trng trong Windows

  1. Tm Environment Variables trong thanh tm kim chn Edit the system environment variables.
  2. Trong hp thoi System Properties Tab Advanced Bm nt Environment Variables.
  3. Nhn New ti User variables to bin mi trng, nhp tn bin v gi tr.
In[1]:
# S dng bin mi trng
import os

# os.environ['var'] = '1234567890'

env = os.getenv('var')
print(env)

1234567890

C th kim tra xem bin mi trng c trong h thng cha bng Command Prompt vi lnh echo %var%. Kt qu s tr v gi tr c t trong h thng.

S dng file .env (file cha bin mi trng)

.env file cha bin mi trng ty chnh cho mi d n, khng cn ci t trong h thng. Gip qun l linh hot hn v ty bin cho tng d n.

Kt hp cng .gitignore v th vin (python-dotenv & os) qun l cc bin mi trng.

  1. To file .env
  2. Dng th vin gi file .env v ly gi tr

env = """
API_KEY=your_api_key_123456
DATABASE_PASSWORD=your_db_password_abcdef
SECRET_TOKEN=my_secret_token_7890

    f.write(env.strip())

from dotenv import load_dotenv
import os

load_dotenv()  # T ng c file .env cng th mc vi notebook

# Ly bin mi trng ra s dng
api_key = os.getenv('API_KEY')
db_password = os.getenv('DATABASE_PASSWORD')
secret_token = os.getenv('SECRET_TOKEN')

print("API_KEY:", api_key)
print("DATABASE_PASSWORD:", db_password)
print("SECRET_TOKEN:", secret_token)

DATABASE_PASSWORD: your_db_password_abcdef
