  • Github

    • Nền tảng lưu trữ và quản lý mã nguồn

    • Trang web cung cấp các tài liệu và hướng dẫn cho lĩnh vực công nghệ thông tin nói chung và khoa học dữ liệu nói riêng. W3schools có các hướng dẫn về: ngôn ngữ lập trình, lập trình web, cơ sở dữ liệu, ngôn ngữ truy vấn, công cụ và các thư viện trong lĩnh vực KHDL.

Trang web dành cho khoa học dữ liệu

  • Kaggle

    • Nền tảng trực tuyến nối tiếng dành cho khoa học dữ liệu, Kaggle cung cấp các khóa học hướng dẫn lập trình, bộ dữ liệu, các cuộc thi, cộng đồng lớn về khoa học dữ liệu và lập trình notebook(.ipynb) có tích hợp GPU để train được các mô hình trong lĩnh vực Deeplearning nhanh chóng.

  • Google Colab

    • Nền tảng viết và chạy code python trên nền web của Google, được thiết kế đặc biệt cho các tác vụ liên quan đến khoa học dữ liệu.
